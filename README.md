# Hogwarts Project – Prisoner of Azkaban

## General Presentation
This project is a text-based interactive adventure game developed in Python and inspired by the Harry Potter universe.
The player creates a wizard character and progresses through multiple chapters at Hogwarts.
Chapter 4 is inspired by *Harry Potter and the Prisoner of Azkaban* and focuses on themes such as Dementors, the Marauder’s Map, and a werewolf encounter.

The game uses user choices, randomness, and character attributes to create different outcomes.
